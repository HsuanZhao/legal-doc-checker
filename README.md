# 📄 法律文书低级错误检查器 (Legal Doc Checker)

> 律师日常工作中的“最后一道防线”——快速扫描法律文书中的常见低级错误，避免因笔误、格式、法条引用等问题损害专业形象。

## 🚀 简介

法律文书低级错误检查器是一个专为法律从业者设计的 **Claude Skill**。它能够自动扫描法律文书中的六大类常见错误，包括错别字与易混词、法条引用错误、格式规范错误、事实与数据错误、表达与语法错误以及法律概念错误，并生成结构化的错误报告。

该 Skill 基于最高人民法院《人民法院民事裁判文书制作规范》、天同律师事务所《法律文书校对要点清单》以及 80+ 例法律写作易错词库开发，可帮助律师、法务、书记员等在提交文书前完成最后一道质量把关。

## ✨ 功能特点

- **6 大类 80+ 检查点**：覆盖错别字、法条引用、格式、事实数据、表达语法、法律概念
- **🎯 精准定位**：标注错误所在段落、原文内容及修改建议
- **⚠️ 优先级分级**：区分严重错误、一般错误和格式问题，便于优先处理
- **📚 基于权威规范**：参考最高法文书制作规范及多家律所校对清单
- **🔊 可触发式调用**：通过自然语言触发，无需额外配置

## 🎯 适用场景

- 起诉状、答辩状、代理词、法律意见书等诉讼文书
- 合同、协议、律师函等非诉讼文书
- 法院裁判文书、仲裁裁决书校对
- 法律实习生、初级律师的自检工具

## 🛠️ 使用方法

### 1. 安装 Skill

将 `SKILL.md` 文件放置在 Claude 或其他支持 Skill 功能的 AI 客户端的指定目录中（具体路径请参考对应客户端说明）。

### 2. 触发检查

在与 AI 对话时，使用以下任一触发词：

- `检查文书`
- `法律文书校对`
- `文书错误`
- `低级错误检查`
- `法律写作自检`

### 3. 提供文书

将需要检查的法律文书全文粘贴在触发词之后，AI 将自动分析并输出结构化的错误报告。

### 4. 获取报告

输出格式示例：

```
text
【错误报告】

▶ 严重错误（可能导致法律效力问题）
[1] 第3段：法条名称错误
    原文："根据《合同法》第60条..."
    建议："根据《民法典》合同编第60条..."
    分类：法条引用错误

▶ 一般错误（影响专业形象）
[2] 第7段：易混词误用
    原文："被告支付订金5万元"
    建议："被告支付定金5万元"
    分类：错别字与易混词

▶ 格式/表达问题
...

────────────────────
总计：严重错误X个，一般错误X个，格式问题X个
建议：优先处理严重错误...
```


## ⚠️ 局限性

本 Skill 无法替代人工审查，其局限性包括：

- ❌ **无法验证事实真实性**：仅检查文书内部一致性，不核实所述事实是否客观
- ❌ **无法判断法律适用**：不评估法律选择是否正确、诉讼策略是否得当
- ❌ **无法识别逻辑谬误**：不检查论证逻辑或推理链条的合理性
- ❌ **依赖输入准确性**：若文书来自 OCR 识别，可能存在原文错误（需用户自行确认）
- 📅 **基于历史规范**：知识截止于 2026 年，不包含此后修订的法律法规
